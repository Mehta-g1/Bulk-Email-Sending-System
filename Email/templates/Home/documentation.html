Bulk Email Sending System

Bulk Email Sending System designed to streamline the process of sending personalized emails to a large number of recipients. It provides a user-friendly interface for managing users, creating email templates, handling recipient lists, and sending emails efficiently.

Built with the powerful Django framework, it ensures security, scalability, and maintainability. The system is ideal for marketing campaigns, newsletters, notifications, and any scenario requiring mass email distribution.

Key Features

  • User Authentication: Secure user registration, login, and session management.
  • Dashboard Analytics: Real-time tracking of sent, failed, opened, and clicked emails.
  • Campaign Management: Group email sending operations into campaigns for better tracking.
  • REST API: Programmatic access to send emails and manage data.
  • OTP Service: Integrated One-Time Password verification for security.
  • Template Management: TinyMCE rich-text editor with dynamic placeholders.

API Reference

Integrate our platform into your own applications using our secure REST API.

Authentication

All API requests must include your API key in the Authorization header as a Bearer token.

Authorization: Bearer

You can generate your API key from the Account Settings page.

Endpoints

1. Send Single Email

URL: POST /api/send-email/

Request Body:

{
  "to_email": "recipient@example.com",
  "subject": "Greetings",
  "body": "
Hello!
Sent via API.
"
}
2. Create Bulk Campaign

URL: POST /api/send-bulk/

Request Body:

{
  "recipient_ids": [12, 15, 23],
  "campaign_name": "API Marketing Campaign"
}
3. OTP Service (Public)

Send and verify OTPs without an API key.

  • POST /api/send-otp/ - Payload: {"email": "..."}
  • POST /api/verify-otp/ - Payload: {"email": "...", "otp": "..."}

Response Format

All responses are returned in JSON format.

{
  "success": true,
  "message": "Operation successful",
  "data": { ... }
}

JavaScript Example (Fetch)

fetch('/api/send-email/', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    to_email: 'recipient@example.com',
    subject: 'API Test',
    body: '<p>Hello!</p>'
  })
})
.then(res => res.json())
.then(data => console.log(data));

User Manual

Follow these steps to start sending emails:

  1. Configure SMTP: Go to Account Settings and enter your SMTP details (host, port, app password).
  2. Create Template: Use the Template builder to craft your email message.
  3. Add Recipients: Import recipients via CSV or add them manually.
  4. Send Campaign: Go to the Dashboard, select recipients, and click "Send Campaign".

Bulk Add (CSV/XLSX)

Structure your file with these columns:

email,name,receipent_category,source,comment

App Passwords

Use an App Password instead of your regular Gmail/Outlook password for security.
