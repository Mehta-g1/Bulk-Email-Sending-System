💡 Tip: Use an App Password for Better Security

For services like Gmail or Outlook that use 2-Factor Authentication (2FA), it is strongly recommended to use an App Password instead of your regular account password. An App Password is a unique 16-digit code that gives this application permission to send emails from your account without exposing your main password.

API Integration Guide

Include your API key in the Authorization header for all requests: 
Authorization: Bearer YOUR_API_KEY

Endpoint: POST /api/send-email/
Payload:
{
  "to_email": "recipient@example.com",
  "subject": "Hello",
  "body": "<h1>HTML Content</h1>"
}

  • GET /api/campaigns/ - List all
  • GET /api/stats/ - Usage stats
  • GET /api/logs/ - View history

Send 6-digit verification codes to any email.
POST /api/send-otp/
{"email": "user@test.com"}

Example of sending an email using JS: 
fetch('/api/send-email/', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    to_email: 'test@example.com',
    subject: 'API Test',
    body: '<p>Hello from JS!</p>'
  })
})
.then(res => res.json())
.then(data => console.log(data));
